FAQ

Frequently asked questions

What is Model Context Protocol (MCP)?
Model Context Protocol is an open standard for enabling secure, contextual communication between AI models and external data sources. It allows AI systems to access tools, databases, and APIs while maintaining security boundaries.
How does MCPVault ensure compliance?
MCPVault automatically implements security controls required for SOC2, HIPAA, ISO27001, and other frameworks. We provide continuous monitoring, audit logging, access controls, and automated compliance reporting to ensure your AI infrastructure meets regulatory requirements.
Can MCPVault work with existing MCP servers?
Yes! MCPVault acts as a secure proxy layer that works with any MCP-compliant server. Simply point your existing servers to MCPVault, and we'll handle authentication, encryption, monitoring, and compliance without requiring any code changes.
What kind of monitoring does MCPVault provide?
MCPVault offers real-time monitoring of all MCP server interactions, including request/response tracking, performance metrics, security alerts, and anomaly detection. Our dashboard provides insights into usage patterns, potential threats, and compliance status.
How quickly can we get started?
Most teams can deploy MCPVault in under 30 minutes. Our CLI tool automatically discovers MCP servers in your environment, configures security policies, and starts monitoring immediately. Enterprise deployments with custom requirements typically take 1-2 days.
